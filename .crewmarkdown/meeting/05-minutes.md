# Meeting: Minutes — Full Meeting Record

## Need
- Agenda from convene step
- Brainstorm output
- Decision records
- Action items

## Instructions

### 1. Compile Full Minutes
1. Combine all meeting artifacts into a single comprehensive record:
   - Header: title, date, time, duration, objective
   - Attendees list with facilitator
   - Agenda as planned
   - Discussion summary per agenda item
   - Decisions with rationale and alternatives
   - Action items table
   - Parking lot
   - Next meeting suggestions

### 2. Log Minutes
1. Write minutes to `.crewmarkdown/logs/<yyyy>/<MM>/<dd>/<HHmmss>/<chat-slug>-[<HHmmss>]/meeting-minutes.md` (derive `<chat-slug>` from meeting topic)
2. Create directory structure: `.crewmarkdown/logs/<yyyy>/<MM>/<dd>/<HHmmss>/<chat-slug>-[<HHmmss>]/`

### 3. Update State
1. Update `.crewmarkdown/state/workflow.json` with meeting outcomes

### 4. Summary for User
1. Present concise summary of decisions and action items

## Done

- Minutes archived to log directory
