Why CrewMarkdown?

AI agents are great at writing code but terrible at software process. Without structure, every prompt is a gamble — you might get code with no tests, no threat model, no PRD, no deploy plan.

CrewMarkdown gives AI agents a repeatable software engineering brain. 33 roles + 58 sub-specializations with hardened contracts, workflows, and quality gates. Drop it in any project, point your AI at it, and get production-grade results.

No SaaS. No API keys. No vendor lock. Just markdown files any AI agent can read.

Quick Start

# 1. Copy to your project
cp -r .crewmarkdown ./your-project/

# 2. Generate codebase map
./.crewmarkdown/scripts/generate-codebase-map.ps1

# 3. Configure AI agents
./.crewmarkdown/config/setup.ps1        # Windows
./.crewmarkdown/config/setup.sh --all   # Linux/macOS

# 4. Make a request
#    "PM, write a PRD for login"
#    "Squad (QA, Security), verify quality"
#    "Build the full e-commerce platform"

Your AI agent reads .crewmarkdown/00-objectives.md, routes to the right objective, loads the squad, and executes.
