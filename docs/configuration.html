Setup Script

The easiest way to configure AI agents for your project:

PowerShell (Windows) 
./.crewmarkdown/config/setup.ps1
Bash (Linux/macOS) 
./.crewmarkdown/config/setup.sh --all

Manual Setup

opencode

{
  "skills": [".crewmarkdown/skills"],
  "commands": [
    {
      "name": "CrewMarkdown",
      "description": "Run CrewMarkdown workflow",
      "command": "opencode --agent .crewmarkdown/roles/orchestrator/contract.md"
    }
  ]
}

Other Tools

  • Cursor: Copy .crewmarkdown/config/cursor.rules to .cursor/rules
  • Claude Code: Copy .crewmarkdown/config/claude.MD to CLAUDE.md
  • GitHub Copilot: Copy .crewmarkdown/config/copilot.instructions to .github/copilot-instructions.md

Light Variant

A compressed 7-step variant for MVPs, prototypes, and solo development.

Full Light
When Production systems, teams MVPs, solo dev, rapid experiments
Objectives 11 with full squads 7 compressed steps
Roles 33 specialized roles Reduced role set
Ceremony Security gates, ADRs, debate Minimal documentation
Location .crewmarkdown/ (default) .crewmarkdown/light/

Switch any time. Start with Light → move to Full as complexity grows.
