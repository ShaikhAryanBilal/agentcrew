# Debate Rounds — The Structured Debate

## Need
- Panel assembled with stances from previous step
- Facilitator + timebox confirmed
- Decision question + options clear

## Round Structure

```
┌─────────────────────────────────────────────────────────┐
│                  ROUND STRUCTURE                         │
├─────────────────────────────────────────────────────────┤
│ Round 1: Opening Statements (2 min each)               │
│   Each panelist presents their stance + top 2 reasons   │
│                                                         │
│ Round 2: Cross-Examination (3 min per pair)            │
│   Panelists challenge each other's assumptions          │
│   Facilitator directs Q&A between opposing viewpoints   │
│                                                         │
│ Round 3: Rebuttal + Closing (2 min each)               │
│   Each panelist addresses counter-arguments             │
│   Each panelist gives final recommendation              │
└─────────────────────────────────────────────────────────┘
```

Light debate: Round 1 + Round 3 only. Skip cross-examination.
Standard debate: All 3 rounds.
Full debate: All 3 rounds + optional expert testimony.

## Do

### 1. Round 1 — Opening Statements

- [ ] Facilitator reads the decision question aloud
- [ ] Each panelist gets 2 min (or N min based on format)
- [ ] Format: "As [role], I recommend Option [N] because [reason 1] and [reason 2]."
- [ ] No interruptions. Facilitator enforces timebox.
- [ ] Scribe captures each statement verbatim key points

Opening statement template:
```
As [role], I recommend [Option N].
Top reasons:
1. [reason with evidence]
2. [reason with evidence]
Risk I've considered: [risk]
```

### 2. Round 2 — Cross-Examination

- [ ] Facilitator identifies the sharpest disagreement from Round 1
- [ ] Questions directed: panelist A asks panelist B (opposing view)
- [ ] Format:
  ```
  Q: [Panelist A] to [Panelist B]: "Your option assumes [X]. What if [X] is false?"
  A: [Panelist B responds]
  ```
- [ ] Facilitator ensures:
  - Questions are about the option, not the person
  - No filibustering — answers must be direct
  - Equal speaking time across panelists
- [ ] Each exchange: 3 min max, then rotate
- [ ] Scribe captures key exchanges

Cross-examination focus areas:
```
| Focus Area | Question Pattern |
|------------|-----------------|
| Assumption | "Your option assumes X. What if X doesn't hold?" |
| Trade-off | "You gain X but lose Y. Why is X worth losing Y?" |
| Evidence | "What data supports your claim that X is faster?" |
| Risk | "You mentioned risk R. How do you mitigate it?" |
| Scale | "Does your option work at 10x scale? 100x?" |
```

### 3. Round 3 — Rebuttal + Closing

- [ ] Each panelist gets 2 min for final statement
- [ ] Must address counter-arguments heard in Round 2
- [ ] Format:
  ```
  As [role], I've heard the counter-arguments.
  My response to [key challenge]: [evidence/counterpoint]
  My final recommendation remains/shifts to [Option N].
  Bottom line: [1-sentence why this is the right choice]
  ```
- [ ] Panelists may change their recommendation based on evidence
- [ ] Scribe captures final positions

### 4. Facilitator Summary

After all rounds, facilitator summarizes:

```
## Facilitator Summary

### Points of Agreement
- [All panelists agree on: ...]

### Points of Disagreement
- [Key unresolved divergence: ...]

### Evidence Presented
- [Evidence 1: ...]
- [Evidence 2: ...]

### Shifts
- [Panelist X shifted from Option A → B because ...]
```

## Rules of Engagement

| Rule | Why |
|------|-----|
| Attack ideas, not people | Keeps debate constructive |
| Evidence over opinion | "I've seen this fail" > "I don't like it" |
| Role-first framing | "As [role], I need X" not "I want X" |
| One speaker at a time | Facilitator gong if interrupted |
| Timebox is sacred | Better a sharp 2 min than a rambling 10 min |
| No strawman | Represent opposing views accurately before rebutting |

## Decision Recording

During rounds, scribe records:

```
## Debate Transcript

### Round 1: Opening Statements
[time] [role]: [statement]

### Round 2: Cross-Examination
[time] [role A] → [role B]: [question]
[time] [role B]: [answer]

### Round 3: Rebuttal + Closing
[time] [role]: [final statement]
```

## Done
- All rounds completed
- Facilitator summary written
- Scribe transcript captured
- Final positions recorded (including any shifts)
