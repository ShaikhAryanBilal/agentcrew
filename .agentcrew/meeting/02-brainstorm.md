# Meeting: Brainstorm — Ideation + Discussion

## Need
- Agenda set with topic + timeboxes
- Roles assembled and present
- Divergent thinking phase

## Do

### 1. Opening Context
Facilitator presents the topic with context:
- What are we solving?
- Why does it matter?
- What constraints apply? (time, budget, tech, team)
- What's the desired outcome?

### 2. Round-Robin Ideation
Each role contributes from their perspective, one at a time:

| Round | Role Contributes |
|-------|-----------------|
| 1 | PM: Business goals, user needs, priorities |
| 2 | UX: User flows, design considerations, personas |
| 3 | Frontend: UI approach, component considerations |
| 4 | Backend: API needs, data model impact |
| 5 | Tech Lead: Technical constraints, patterns, risks |

**Format:**
```
[Role]: [Idea/Contribution]
- Key point 1
- Key point 2
- Question/Concern
```

### 3. Silent Ideation (Optional)
For complex topics, give each role 2 min to write ideas independently first:
```
--- Silent Ideation (2 min) ---
[Role]'s ideas:
1. ...
2. ...
3. ...
```

Then share and build on each other's ideas.

### 4. Parking Lot
Off-topic but valuable ideas go to the parking lot:
```
## Parking Lot
- [Idea] -> revisit in separate meeting
- [Idea] -> defer to next agenda
```

### 5. Summary
Facilitator summarizes all ideas gathered:

```
## Ideas Generated
1. [Idea] — proposed by [Role]
2. [Idea] — proposed by [Role]
3. [Idea] — proposed by [Role]

## Themes
- Theme A: [related ideas]
- Theme B: [related ideas]

## Parking Lot
- [deferred ideas]
```

## Techniques

### Round-Robin
Each role speaks in turn. No one speaks twice until everyone has spoken once.

### Brainwriting
Roles write ideas silently, then pass to next role to build upon.

### Yes And
Build on ideas before critiquing. "Yes, and we could also..."

### SCAMPER
**S**ubstitute, **C**ombine, **A**dapt, **M**odify, **P**ut to other use, **E**liminate, **R**earrange

## Anti-Patterns

| Don't | Instead |
|-------|---------|
| Evaluate during ideation | Capture first, evaluate later |
| Let one person dominate | Enforce round-robin |
| Dismiss ideas quickly | Park them if not relevant now |
| Skip quiet roles | Specifically invite them |

## Done
- All roles contributed
- Ideas captured in full
- Parking lot recorded
- Summary prepared
