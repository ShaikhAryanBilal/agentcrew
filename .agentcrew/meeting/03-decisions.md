# Meeting: Decisions — Consensus + Resolution

## Need
- Ideas generated from brainstorming
- Decision points identified
- Roles ready to evaluate

## Decision Flow

```
Multiple options for a decision?
├── No → Single path → Record and proceed
└── Yes → Evaluate options:
    ├── Quick consensus? → Record decision
    ├── Mild disagreement? → Consent process (no strong objections)
    ├── Strong disagreement? → Escalate to Debate (debate/trigger)
    └── Need structured eval? → Weighted scoring
```

## Do

### 1. Identify Decision Points
List what needs to be decided:

```
## Decisions Needed
1. [Decision question 1]
2. [Decision question 2]
```

### 2. Present Options
For each decision point, list the options on the table:
```
## Decision: [Question]
### Options
1. [Option A] — proposed by [Role]
   - Pros: [list]
   - Cons: [list]
2. [Option B] — proposed by [Role]
   - Pros: [list]
   - Cons: [list]
```

### 3. Role Perspectives
Each role weighs in from their lens:

```
### [Role] Perspective
My stake: [what matters to this role]
Preference: [Option A / B / other]
Rationale: [why]
Concerns: [what could go wrong]
```

### 4. Reach Decision

**Consensus**: All actively support the decision.
```
Consensus: YES
All roles agree Option A is the best path.
```

**Consent**: No one has a strong objection (even if not everyone's first choice).
```
Consent: YES
No blocking objections to Option A.
Reservations: [Role] notes [concern], but not blocking.
```

**Weighted Scoring**: Use criteria with weights.
```
| Criteria        | Weight | Option A | Option B |
|-----------------|--------|----------|----------|
| Cost            | 20%    | 8        | 6        |
| Timeline        | 25%    | 7        | 8        |
| Quality         | 30%    | 9        | 7        |
| Risk            | 25%    | 6        | 9        |
| **Total**       | 100%   | 7.55     | 7.55     |
```

**Debate**: If strong disagreement persists, escalate:
-> `debate/trigger.md` for structured debate

### 5. Record Decision
```
## Decision Record
- **Question**: [what we decided]
- **Decision**: [chosen option]
- **Rationale**: [why this path]
- **Alternatives Considered**: [other options and why not]
- **Consensus Level**: unanimous / consent / majority / debate-resolved
- **Reservations**: [any minority concerns]
- **Decided by**: [roles involved]
```

## Done
- All decision points resolved
- Each decision recorded with rationale + alternatives
- Any un-resolvable items escalated to debate
- Decision records ready for minutes
