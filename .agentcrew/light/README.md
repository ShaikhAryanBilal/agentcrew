# AgentCrew — Light Variant

## When to use
- MVPs, prototypes, small projects
- Solo dev or 2-person team
- Projects where full SDLC is overkill

## What's included
7 objectives covering the full lifecycle (compressed):
1. Clarify Vision (light) — 1-page PRD, 30 min
2. Design Solution (light) — 1-page sketch, 1 hr
3. Plan Work (light) — task breakdown, 30 min
4. Build Feature (light) — code + tests
5. Verify Quality (light) — smoke test, 1 hr
6. Ship Release (light) — deploy, 30 min
7. Operate & Learn (light) — monitoring, 30 min

## How to use
Read `.agentcrew/light/00-router.md` instead of `.agentcrew/00-objectives.md`.
All roles work solo in light mode — no squad coordination needed.

## Switching variants
- **Light → Full:** When complexity grows, use `.agentcrew/00-objectives.md` instead.
- **Full → Light:** When project is stable, switch to light for maintenance.
